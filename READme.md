# Python Developer Test Case

## Используемые технологии
- SQL
- Python 3.12

### Запуск для Linux/macOS( на Windows с помощью Git Bash):
```bash
git clone git@github.com:abyxez/aiti_guru.git
cd aiti_guru/
make all
```

## Результаты:
- schema.png в корне проекта
- Задания 2.1 и 2.2 в терминале

-----------------
## Пояснения:
- Для визуализации была использована библиотека erdantic.
- На Windows запуск выполняется в Git Bash.
- Тестовой СУБД выбрана SQlite3 для легковесности.
- Схема ERD-визуализации неточно отображает schema.sql, но это сделано для лучшей наглядности.
